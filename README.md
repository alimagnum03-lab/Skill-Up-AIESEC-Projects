# AIESEC SkillUp Program Website

A modern, responsive website for the AIESEC SkillUp Program in Bahrain. Students can learn about the program, explore tracks, and register online with automatic form submission to Google Sheets.

## 🌟 Features

- **Responsive Design** — Works on desktop, tablet, and mobile devices
- **Interactive Navigation** — Easy tab-based navigation between sections
- **Online Registration** — Real-time form submission to Google Sheets
- **Multiple Tracks** — IT (Coding) and Entrepreneurship programs
- **Professional Branding** — AIESEC branding and colors

## 📋 Sections

1. **Home** — Welcome message and program highlights
2. **Project Info** — Detailed information about both tracks and the program timeline
3. **Register** — Sign-up form with automatic Google Sheets integration
